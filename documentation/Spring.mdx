# Spring

`import { Spring, config } from 'react-spring'`

## Basic usage

`Spring` will move data from one state to another.

<Playground style={{ overflow: 'hidden' }}>
  <Spring from={{ number: 0 }} to={{ number: 100 }}>
    {props => <div>{props.number}</div>}
  </Spring>
</Playground>

## Interpolation

It doesn't just handle numbers, it can interpolate pretty much everything: 

* colors (names, rgb, rgba, hsl, hsla)
* absolute lenghts (cm, mm, in, px, pt, pc)
* relative lengths (em, ex, ch, rem, vw, vh, vmin, vmax, %)
* angles (deg, rad, grad, turn)
* flex and grid units (fr, etc)
* all HTML attributes
* SVG paths (as long as the number of points matches, otherwise use [custom interpolation](https://codesandbox.io/embed/lwpkp46om))
* arrays
* string patterns (`transform`, `border`, `boxShadow`, etc)
* [`auto` is valid](https://codesandbox.io/embed/pmjomxn60)
* non-animatable string values (`visibility`, `pointerEvents`, etc) 

<Playground style={{  }}>
  <Spring
    from={{
      display: 'inline-block',
      whiteSpace: 'nowrap',
      overflow: 'hidden',
      width: 0,
      background: '#000',
      transform: 'translate3d(100px,0,0) scale(2) rotate(90deg)',
    }}
    to={{ width: 'auto', background: '#ea567c', transform: 'translate3d(0px,0,0) scale(1) rotate(0deg)' }}
    config={config.molasses}>
    {props => <div style={props}>hi there</div>}
  </Spring>
</Playground>
